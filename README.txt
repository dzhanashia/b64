Функции:

isb64(s, log=True)
Проверяет является ли s (string/file) b64.
Возвращает True если является, False иначе.
log определяет включено ли логирование во время выполнения функции (по умолчанию True)

pick_out(s, index=True)
Делит s (string/file) на b64 и нет.
Возвращает индексы b64 слов если index=True, иначе 2 списка - список с b64 и список с остальными словами (по умолчанию True)

delete_clfs()
Удаляет файлы с обученными классификаторами
