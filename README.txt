	
-dataset.txt - набор из слов на английском, аббревиатур и дат, половина из которых закодированы, используемый для обучения и тестирования.
-В программе проверяется качество работы нескольких классификаторов из библиотеки sklearn. Незакомментироваными оставлены те из классификаторов, которые являются наиболее быстрыми из показывающих самые высокие результаты.
-Обучение происходит на признаках: 
1) отношение знаков в нижнем регистре к знакам всего
2) отношение количества цифр к знакам всего
3) количество "="
